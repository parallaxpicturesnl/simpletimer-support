Simple Timer and Countdown

Support page for the Simple Timer and Countdown iOS app.

How It Works

Simple Timer and Countdown helps you prepare for events by creating chains of countdown timers anchored to a single event time. Change the event once and all your timers update automatically.

Frequently Asked Questions

How do I create an event?

Tap the Events tab, then tap the + button in the top right corner. Enter your event name, date, and time. Then add offset timers relative to that event.

How do offset timers work?

Each timer is set as an offset from your event. For example, "2 hours before" means the timer triggers 2 hours before your event time. If you change the event time, all timers automatically recalculate.

What is SimpleTimer Pro?

SimpleTimer Pro is a one-time purchase that unlocks unlimited events, unlimited timers, unlimited templates, calendar import, iCloud sharing, and all widget sizes.

How do I restore my purchase?

Go to Settings and tap "Restore Purchases". Make sure you are signed in with the same Apple ID you used for the original purchase.

How do notifications work?

Each timer can have a notification. When the countdown reaches zero, you will receive an alert. Make sure notifications are enabled in Settings.
